Patchwork

Getting Started

Requirements

Patchwork requires PHP 5.3.0 or higher to run. Furthermore, no opcode caches should be enabled when using Patchwork.

Importing Patchwork

This step seems quite straightforward, as all you have to do is include Patchwork.php:

require __DIR__ . "/patchwork/Patchwork.php";

However, the exact place from where Patchwork is included makes a great difference. The general rule is to do so at the very starting point of your application.

This is explained by the fact that Patchwork is unable to redefine any functions that are defined in files included earlier than Patchwork.php itself. These also include the file from which Patchwork.php is included, so any functions defined in it will not be eligible for redefinition either.

Further Steps

You can already start redefining functions now:

Patchwork\replace("Post::delete", function()
{
    echo "No, sorry. Won't do that. This post deserves to live.";
});

While Patchwork barely gets any more difficult than this, you might still want to study some more examples before starting off.
