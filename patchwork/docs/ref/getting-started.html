Requirements

Patchwork requires PHP 5.3.0 or higher to run. Furthermore, no opcode caches should be enabled when using Patchwork.

Importing Patchwork

This step seems straightforward: all you have to do is include Patchwork.php. However, it must be included before all the files that define any of the functions and methods which may have to be redefined. Therefore, it is recommended to do so at the very entry point of your application.

# Include Patchwork
require __DIR__ . "/patchwork/Patchwork.php";

# Include everything else

Further Steps

Now, you may start redefining functions:

Patchwork\replace("Post::delete", function()
{
    die("Why would you want to do that?");
});

For more details, check out the Redefinition API reference.
