Patchwork

Requirements

PHP Version

Patchwork requires at least version 5.3.0 of PHP. If you're unsure which version of PHP you're running, you can check it by running php -v from the command line or checking the value of the PHP_VERSION constant.

Opcode Caches

Patchwork hasn't been tested with any opcode cache, such as APC, XCache or eAccelerator. However, since Patchwork preprocesses PHP code, it has been proposed that these caches might cause Patchwork to behave improperly by bypassing the preprocessing mechanism. Therefore, caution must be taken when using Patchwork with an opcode cache enabled.
