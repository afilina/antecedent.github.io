Patchwork

Requirements

PHP Version

Patchwork requires at least version 5.3.0 of Zend's PHP. If you're unsure which version of PHP you're running, you can check it by running php -v from the command line or evaluating the PHP_VERSION constant.

Opcode Caches and HHVM

Patchwork might break when an opcode cache (OPCache, APC etc.) is in use; the same also goes for running it on HHVM.

This claim, however, has been neither verified nor falsified by the developer of Patchwork; that is planned for near future. In the meantime, you are encouraged to experiment and report your findings.
