Patchwork

Function redefinition and call interception for PHP. No extensions required.
For creating test doubles of nearly anything, including static, private and internal callables,
as well as for debugging, profiling and aspect-oriented programming.

function size($x)
{
    return count($x);
}

size(array(1, 2)); # => 2

1. Define a function

use Patchwork as p;

p\replace("size", function($x)
{
    return "huge";
});

size(array(1, 2)); # => "huge"

2. Replace its definition

p\undoAll();

size(array(1, 2)); # => 2

# Note that all of these steps
# occur at the same runtime!

3. Undo the redefinition

We make the assumption of a responsible developer, of one that will continue writing maintainable code despite the hurdle of testability being lowered, meaning that a major motivator for maintainablity is no longer present.
